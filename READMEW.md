## 🚀 Installation Rapide

1. Décompresse `Wumpus-multitool.zip`
2. Ouvre un terminal dans le dossier
3. Installe les dépendances :

```bash
pip install -r requirements.txt
```

4. Lance le launcher :

```bash
python launch.py
```

## ⚙️ Commandes du Launcher

- Tape le **numéro** de l’outil pour le lancer
- **`G`** → Configurer GitHub Token
- **`C`** → Changer de thème
- **`N`** → Page suivante
- **`P`** → Page précédente
- **`0`** → Quitter
